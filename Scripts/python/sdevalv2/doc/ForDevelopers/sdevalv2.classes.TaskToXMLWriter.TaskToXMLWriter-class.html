createXMLFromTask(self, task, xmlRessources)

source code 

Returns an XML-Representation of the Task. This representation has the following form:

 <task>
   <name>
     "The Taskname"
   </name>
   <computationproblem>
     "The computationproblem associated to the task"
   </computationproblem>
   <computeralgebrasystems>
     <computeralgebrasystem>
       "Name of computer algebra system 1"
     </computeralgebrasystem>
     <computeralgebrasystem>
       "Name of computer algebra system 2"
     </computeralgebrasystem>
     ...
   </computeralgebrasystems>
   <sdtables>
     <sdtable>
       <tablename>
         "Name of the sdtable"
       </tablename>
       <probleminstances>
         <probleminstance>
           "Name of the problem instance"
         </probleminstance>
         ...
       </probleminstances>
     </sdtable>
     ...
   </sdtables>
Parameters:
  • task (Task) - The task we want to have the xml representation of
  • xmlRessources (XMLRessources) - The Folder where we can find the different SD-Tables and where we can check the entries
Returns: xml.dom.minidom.Document
An xml-representation of the task
