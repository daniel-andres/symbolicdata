Class MachineSettings

source code

This class represents the machine settings for the taskfolder. Machine settings are defined in the following way:

Machine settings contain a collection of machine-specific constants; Those are:

  • For every used computer algebra system the command for executing it
  • The command on the target-machine for calculating the time an execution needs (time-command) with the needed options for that time command

Author: Albert Heinle

Returns an XML representation of the machine settings. It has the following form:

 <machinesettings>
   <casdictionary>
     <entry>
       <key>
         "Name of computer algebra system"
       </key>
       <value>
         "Execution command on target machine"
       </value>
     </entry>
     ...
   </casdictionary>
   <othervars>
     <timecommand>
       "The time command"
     </timecommand>
   </othervars>
 </machinesettings>
