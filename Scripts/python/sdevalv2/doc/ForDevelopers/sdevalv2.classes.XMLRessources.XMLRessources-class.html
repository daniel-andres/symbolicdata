Class XMLRessources

source code

This class is the interface to the Folder XMLRessources in the SymbolicData folder tree. Its purpose is, that one can access the different SD-Tables inside this folder.

Author: Albert Heinle

Contact: albert.heinle@rwth-aachen.de
