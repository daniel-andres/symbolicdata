Class ResultingFile

source code

This is a representation of an result file given as an output of the computer algebra systems on the problem instances. A resulting file is always associated to a computer algebra system, a problem instance and its SD-Table where it has been taken from. It consists of the output of the computer algebra system on this problem instance and the output of the time command on the target machine.

Author: Albert Heinle

Contact: albert.heinle@rwth-aachen.de
