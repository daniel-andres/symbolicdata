Class ResultedTimings

source code

This is a decorator class of the Proceedings class. It provides to every entry in the Proceedings lists an extra entry, namely an ResultingFile-Instance, if existent.

See Also: Proceedings

Author: Albert Heinle

Contact: albert.heinle@rwth-aachen.de

(Constructor)

source code 

This is the constructor of the ResultedTimings-Class. Either a proceedings instance is given and this will be used then, or a new Proceedings instance is generated given the task and the timeStamp.

Parameters:
  • proceedings (Proceedings) - The proceedings instance we want to decorate
  • task (Task) - The task we want to create the Proceedings instance of
  • timeStamp (string) - The timestamp of the execution of the task
Raises:
  • IOError - If neither a Proceedings instance nor a task or a timeStamp is given
