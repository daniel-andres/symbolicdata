Class ModPS

source code

The concrete problem instance ModPS from SymbolicData. It represents polynomial systems with coefficients in a finite field. Details on ModPS can be found in SymbolicData.

__init__(self, name, sdTable, vars=None, basis=None, characteristic=None)
(Constructor)

source code 

The constructor of an ModPS-Instance. The name and the SD-Table are needed for the superclass ProblemInstance. The other parameters are the vars used for the polynomial system, the basis containing polynomials and the characteristic of the basedomain.

Parameters:
  • name (string) - The name of the ModPS-ProblemInstance
  • sdTable (SDTable) - The SDTable containing ModPS-Instances. This input will not be checked.
  • vars (list) - A list of variables used in the INTP-System
  • basis (list) - The polynomials forming a basis of the INTP-System. This input will not be checked whether there are polynomials using variables not in the list of variables.
  • characteristic (unsigned int, prime-power) - The characteristic of the basedomain
