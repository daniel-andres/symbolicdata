Class Proceedings

source code


This is a class representation of the proceedings.

Proceedings contain information about the status of the execution of the files in the
taskfolder.
The status can be one of the following options:
  1) RUNNING
  2) WAITING
  3) COMPLETED
  4) ERROR
The meanings of the status can be derived from their names. During the execution, we additionally
write an html file with the visualization of the proceedings.
