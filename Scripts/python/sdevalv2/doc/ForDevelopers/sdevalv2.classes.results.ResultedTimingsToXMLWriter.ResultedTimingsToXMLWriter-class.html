createXMLFromResultedTimings(self, resultedTimings)

source code 

Returns an XML-Representation of the ResultedTimings. This representation has the following form:

 <proceedings>
   <timestamp>
     "The timestamp"
   </timestamp>
   <task>
     "the name of the executed task"
   </task>
   <running>
     <entry>
       <probleminstance>
         "A problem instance"
       </probleminstance>
       <computeralgebrasystem>
         "A computer algebra system"
       </computeralgebrasystem>
     </entry>
   </running>
   <waiting>
     "same as running"
   </waiting>
   <completed>
     "same as running, but with"
      <timings>
         <real>
           "real time"
         </real>
         <user>
           "user time"
         </user>
         <sys>
           "sys time"
         <sys>
       </timings>
   </completed>
   <error>
     "same as completed"
   </error>
 </proceedings>
Parameters:
  • resultedTimings (Proceedings) - The proceedings we want to have the xml representation of
Returns: xml.dom.minidom.Document
An xml-representation of the proceedings
