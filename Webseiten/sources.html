Downloading the SymbolicData Sources

The Symbolicdata Tools and data are available via anonymous CVS from the project's CVS repository

cvs -d :pserver:anonymous@pcai003.informatik.uni-leipzig.de:/home/symbolicdata/cvsroot checkout [Subproject]

As subprojects are available

  • SD-1 - the original tools and data as developed during 200-2003. Note that the format is deprecated and will no more be supported.
  • SD-2 - tools and data containing the XMLData (the full data collection as XSchema based XML data) and OWLData (a first translation to OWL standard) data collections and the Scripts directory with best practise sample scripts supplied by different users.
  • Publications - sources of publications, presentations, and the sources of the web site.

Project members can get also write access to the repository. For details please contact Hans-Gert Graebe.
